# 2d-traffic-animation-cpp

This project is a simple 2D animated simulation created in C++ using the graphics.h library.
The animation presents a short story of a character traveling from home to work by car, passing traffic lights and experiencing city traffic.

## Features

- animated character
- animated car
- traffic lights (red / yellow / green)
- house and workplace scenes
- moving sun animation


## Project Structure
CODE – source code of the application  
DEMO – video of the animation


## Technologies
- C++
